# Producto 1 — Diseño UML

## Descripción
En este producto desarrollamos los diagramas estáticos y de comportamiento a partir de la identificación de los requisitos del proyecto. Serán la base para desarrollar e implementar la aplicación.

## Objetivo
Realizar el diagrama de casos de uso y el diagrama de clases para representar el comportamiento y el diseño estático del sistema.

## Entregables (ZIP)
- Diagramas consensuados: casos de uso + clases
- Documento justificativo del consenso + autoevaluación (0–100)
- Ficheros Java: clases del diagrama (atributos, constructores, getters/setters, toString)

- `/Producto-1/Diagramas/`
- `/Producto-1/Documento/`
- `/Producto-1/Java/`
